## 1. Deskripsi Singkat Project
Project ini merupakan implementasi berbagai teknik manipulasi citra digital menggunakan Python dan OpenCV. Eksperimen dilakukan untuk memahami konsep dasar pengolahan citra digital serta menganalisis pengaruh perubahan nilai piksel terhadap tampilan visual citra, tingkat kecerahan (*brightness*), kontras (*contrast*), dan detail objek.

Teknik manipulasi citra yang diterapkan pada project ini meliputi:
- Membaca dan menampilkan citra digital
- Konversi format warna BGR ke RGB
- Penyesuaian ukuran citra (*Resize*)
- Penggabungan dua citra menggunakan *Image Blending*
- Pembuatan citra negatif (*Image Negative*)
- Transformasi Logaritmik (*Log Transformation*)
- Transformasi Gamma (*Gamma Correction*)
